“基岩时代”添加了很多与“基岩”有关的物品。例如“基岩锭”，“基岩粒”，“基岩甲”，“基岩护腿”，“基岩头盔”，“基岩靴”和“磨制基岩块”。

### 常见问题：

**1、如何获取基岩锭**



在高度为-30至-64的地方可以遇到基岩矿

可以使用钻石镐或下界合金搞挖掘，这会掉落1-3个基岩粒



当然，如果你为你的稿子附魔了精准采集，那就会掉落1整个基岩矿



9个基岩粒可以合成1个基岩锭，反过来同理



你也可以把整个基岩矿放进高炉里烧制，这会烧制成1个基岩锭

**注意，一定要是高炉，熔炉和烟熏炉都不行**
